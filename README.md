## Web Component Examples

./index.html links to all 3 web component examples

#### Native
- Code in build-components/native/user-card.js
    
#### Stencil
- To build:
    1. cd into build-components/stencil
    2. run ``npm install``
    3. run ``npm run build``
    
#### Angular
- To build:
    1. cd into build-components/angular
    2. run ``npm install``
    3. run ``npm run build``
